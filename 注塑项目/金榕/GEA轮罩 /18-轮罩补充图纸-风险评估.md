## 二、🔴 高风险 / 必须处理

### 1. 颜色 + 表面(晒纹)未定 —— **需核对**
- 图纸标题栏 Surface Protection = `/`；BOM 里气坝 **Color = TBD、Surface finish = TBD**。
- **材料已定**：Santoprene 103-40 / CELANESE / PMR UB02b、UB05d(Part List 明确，报价按 ¥35/kg 已锁定，无需再改)。
- **待核对**：① 颜色 ② 晒纹规格(纹理编号 / VDI / 皮纹深度)。
- 影响：晒纹规格直接决定模具晒纹费；颜色若需特殊色粉/UV 会影响材料成本。报价按"外观件含晒纹"已留口径，但规格定下来前不能最终锁价。

### 2. NOTE 4：标识必须"模内成型(MOLDING)"
- 要求模内刻字：零件名、材料信息、零件号、**生产日期时钟(Month/Year CLOCK)**、Renault logo(≥10mm)、Renault emblem(≥5mm)。
- 4.3.3 / 4.3.4：**不许出现供应商零件号、不许有可识别供应商的信息(含二维码)**。
- 影响：模具内需做全部刻字 + **可换"日期时钟"镶件**。→ 确认 **29 万/副已包含日期时钟镶件和全部模内刻字**，否则漏报。

### 3. 气坝带卡扣/倒扣，且与多个"非我方供货"件配合
- 剖面 B/C/D/E/H 显示气坝有**一体式卡扣/钩子(倒扣)**，与 `CAD6608276039`(毡本体)、`622210696R`、`628231053R`、`631130139R`、`620253655R`、`CAD6608426546` 咬合。
- 影响 a：倒扣 → 模具需**滑块/斜顶**，确认"一副模"含这些机构及对应成型周期(现按 50s / 360T 估)。
- 影响 b：配合件均非我方供货，配合好坏取决于客户 CAD。→ 协议里写明"配合面/卡扣以客户 3D 数模为准，装配间隙不由我方单独担责"。

### 4. 报价"实验/检测费 = 0"，但图纸有法规与 DVP 要求
- NOTE 1 引用 5 个雷诺标准：**R049 内饰阻燃、R054 外部噪音、R063 挡泥/轮罩、R064 报废回收、R071 外部凸出物**；NOTE 2.8"其余性能见 DVP"。
- 影响：大概率需材料认证(阻燃 R049、回收 R064)+ DVP 试验。我方实验费报 0 → 需补进去，或在协议里明确由客户承担 PPAP/材料认证/DVP。

## 三、🟡 中风险 / 需要留意

### 5. "图上没标的尺寸一律以 3D 数模为准"(NOTE 2.1 / 2.2，修边 ±2mm)
- 光看图不能报价/开模，**必须拿到与图纸版本(001 / 2026-06-10)一致的 STP**。确认我方算体积用的数模就是该版本，别用旧数模。

### 6. NOTE 3 料厚与 BOM 不一致
- NOTE 3.2"GENERAL THICKNESS **3.5mm** +0.5/-1"、3.3"固定点/修边区 2mm"；BOM/Part List 料厚 **2.5mm**。
- 判断：NOTE 3 针对 **PP-PET 毡本体**(压制)，不是气坝；不影响我方(按实测体积算重)。口径与客户对齐即可。

### 7. 左右件对称性(NOTE 2.7)表述含糊且引用了不属于本图的零件号
- NOTE 2.7 出现 `638453927R / 638443963R` 等**非本套零件号**，明显是从别的图复制未改。
- 无法确认 6 个气坝是纯镜像还是"部分不对称"。→ 请客户**书面确认 LH/RH 是否完全镜像**(影响"1+1 一副模"左右型腔特征)。

### 8. 装配间隙 / 平面度
- 前轮罩 与 Air Curtain 间隙 1.5，公差 **+1.5/-1.5**；后轮罩另有 Rear Door Protector 间隙 0 / +0.7。
- TPV 软料 + 平面度要求 1.0，装配间隙不易稳定控制和测量。→ 确认检具与测量方法，避免 PPAP 卡尺寸。

5. 提供与图纸 **001 版一致的 3D 数模** + 周边配合件(620253655R、631130139R 等)数模，明确卡扣配合责任。
