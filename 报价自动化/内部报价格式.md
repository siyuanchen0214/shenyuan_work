# 内部报价格式 · 通用化说明

> 模板文件：`注塑项目/金榕/产品报价内部格式.xlsx`（只读，勿改）
> 本文件是对该模板的字段拆解，供自动填表程序使用。

---

## 前置：开始填表前必须拿到的两样东西

| # | 数据 | 怎么拿 |
|---|---|---|
| 1 | **材料单价** | 先查 `材料价格.md`；库里没有的料 → 联系供应商询价，要求对方出书面报价（POC） |
| 2 | **模具价格** | 直接向模具厂询价，等对方回复后才能填 AJ 列 |

> 这两项拿不到，报价表就不能开始填。其他所有数据都可以从客户图纸/Part List 里提取。

---

## 一、必填字段（每次人工提供）

| 列 | 字段名 | 来源 |
|---|---|---|
| B | 车型 | 客户询价 |
| C | 零件号 | 客户 Part List / 图纸 |
| D | 产品名称 | 客户 Part List / 图纸 |
| G | 模穴数 | 模具方案 |
| H | 取数 | 模具方案 |
| K | 总量纲（万件） | 客户 Part List |
| L | 生命周期（年） | 客户要求，通常 3 年 |
| N | 制品重量（g） | 图纸 / BOM |
| O | 料头重量（g） | 工艺估算，约制品 10–20% |
| T | 材料单价（元/kg） | ★ 先查 `材料价格.md`；**库里没有的，必须向供应商询价，许经理须附 POC（报价截图/邮件）** |
| W | 成形机吨位（T） | 工艺选机 |
| X | 加工时间（s） | 工艺估算 |
| Y | 机台费率（元/min） | 查 `机台费率.md` |
| AE | 运输费（元/件） | 项目级，通常用公式：总价 × 7% |
| AG | 包装费（元/件） | 项目级，通常用公式：总价 × 5% |
| AJ | 模具费（元） | ★ **必须向模具厂询价获得，不可估算** |

---

## 二、公式固定列（程序直接写公式，不需人工输入）

| 列 | 字段名 | 公式（row 用实际行号替换） |
|---|---|---|
| M | 月批量 | `=ROUND(K{r}/L{r}/12*10000,-2)` |
| S | 使用重量 | `=(N{r}+O{r}*(1-P{r})/H{r})*(1+Q{r})+R{r}/M{r}` |
| U | 不良损耗 | `0.01`（固定值） |
| V | 部品材料费 | `=(S{r}*T{r}/1000)*(1+U{r})` |
| Z | 分钟机台费率 | `=Y{r}/60/0.9` |
| AA | 加工不良损耗 | `=U{r}` |
| AB | 加工费 | `=(Z{r}*X{r}/H{r})*(1+AA{r})` |
| AC | 材料管理费 | `=V{r}*0.03` |
| AD | 加工管理费 | `=AB{r}*0.08` |
| AI | 利润 | `=(V{r}+AB{r}+AC{r}+AD{r}+AE{r}+AG{r})*0.08` |
| AK | 模具分摊 | `=AJ{r}/总分摊量`（总分摊量 = 生命周期 × 年量，项目级） |
| AL | 总计未税 | `=SUM(V{r}+AB{r}+AC{r}+AD{r}+AE{r}+AG{r}+AI{r}+AK{r})` |

---

## 三、运输/包装费的循环引用解法

AE（运输 7%）和 AG（包装 5%）如果直接写成 `=AL*7%`，会产生循环引用。
解析解（分母推导自 1 − 7% − 5% − 8%×12%）：

```
分母 = 0.8704

AE = ((V + AB + AC + AD) * 1.08 + AK) / 0.8704 * 7%
AG = ((V + AB + AC + AD) * 1.08 + AK) / 0.8704 * 5%
```
