# 注塑粗略报价规则（Decision Tree）

> 用途：快速估算注塑零件的综合单价，驱动自动报价 agent。
> 结构：三大成本模块独立计算，最终汇总加利润得出报价。

---

## 总公式

```
单件报价 = (材料费 + 模具刀具摊销 + 加工费) × (1 + 利润率)
```

---

## 一、材料价格

> 目标：算出每个零件的材料成本（元/件）

### 1.1 输入条件
- 材料牌号 / 类型
- 零件克重（g）
- 水口料比例（%）

### 1.2 材料识别规则

**从图纸读取材料信息，按以下优先级处理：**

```
情况A：图纸注明材料名称 + 牌号（常见）
  → 例：PC+ABS / MAC5210 / 金发
  → 直接 match 材料价格表，取对应单价

情况B：图纸只注明材料名称，无牌号（少数）
  → 客户需要我们推荐材料
  → 客户会提供物性表（材料物性要求）
  → 根据物性表向厂家询价，不用价格表
  → 报价备注：材料价格待厂家确认

情况C：复合材料（如 PC+ABS）
  → 材料名称字段可能包含多个材料（以 + 或 / 分隔）
  → 逐一 match 价格表中对应牌号
  → 取实际使用牌号的单价
```

**材料价格表匹配逻辑：**
1. 先按完整牌号精确匹配
2. 匹配不到 → 按材料大类模糊匹配，标注"价格待确认"
3. 均无法匹配 → 触发情况B流程

### 1.3 材料单价来源
→ 引用外部表格 `材料价格表.xlsx`（用户定期更新，以最新版为准）

### 1.4 计算公式
```
材料费（元/件）= 净用量（kg）× 材料单价（元/kg）× (1 + 废品率)
净用量 = 单件实际耗料（含水口，从图纸/称重获取）
```

3. 经验估算（规则待积累，见下）

### 1.6 水口料比例

由模具设计（流道布局、浇口数量）和产品特性（外形尺寸、壁厚、进浇位置）共同决定，纯经验判断。

**影响因素：**
- 热流道 → 水口料极少（≈0~2%）
- 冷流道 + 产品小 → 水口料占比高（可达 20~30%）
- 冷流道 + 产品大 → 水口料占比低（5~15%）
- 多穴模 → 流道更长，水口料占比相对更高

## 二、模具刀具费用

> 目标：算出每个零件分摊的模具成本（元/件）

### 2.1 输入条件
- 产品复杂度
- 模具穴数
- 预计总产量（寿命内）
- 是否含热流道

### 2.2 模具总价
→ **外部询价**，拿到供应商报价后填入，不在本系统估算。

### 2.3 摊销公式
```
模具摊销（元/件）= 模具总价（元）÷ 分摊总件数

分摊总件数 = 客户总采购量（件）
           （若采购量 > 模具寿命内最大产量，则取模具寿命内最大产量）
```

**示例（金榕框架前）：**
- 模具总价：21.98 万元，穴数 2，寿命 30 万模
- 客户采购：23 万件/件号 × 2 = 46 万件 < 60 万件（寿命上限）
- 单件摊销：219,766 元 ÷ 460,000 件 = 0.478 元/件

## 三、加工工序费用

> 目标：算出每个零件的生产加工成本（元/件）

### 3.1 设备吨位（锁模力）

**从图纸判断所需机台吨位：**
```
锁模力（T）= 产品投影面积（cm²）× 型腔压力（kg/cm²）÷ 1000 × 安全系数(1.15~1.3)
```
- 型腔压力参考：PP/PE ≈ 250~350 kg/cm²；ABS/PC ≈ 350~500 kg/cm²
- 图纸若直接标注锁模力，优先使用标注值
- 常用吨位段：200T / 320T / 500T / 650T / 850T / 1000T / 1300T / 1600T

### 3.2 成型周期（加工时间）

**影响因素：** 壁厚、材料、模具冷却设计
**经验公式（参考）：**
```
冷却时间（s）≈ 壁厚²（mm）× 2.5（PP基准，其他材料系数不同）
成型周期 ≈ 冷却时间 × 2（注射+保压+开模约占一半）
```
→ **Open Question**：见 `open_questions.md` Q2

### 3.3 班产量
```
班产量（件/班）= 11.5 × 3600 ÷ 成型周期（s）× 穴数
```

### 3.4 各项计算公式

```
单件台时（H）        = 成型周期（s）÷ 3600

加工工时（H）        = 单件台时 × 操作人数 ÷ 稼动率

人力费用（元/件）    = 工时单价 × 加工工时
                     = 24 × 单件台时 × 操作人数 ÷ 稼动率

折旧费用（元/件）    = 设备价格（元）× 0.9
                       ÷ (10年 × 300天 × 2班 × 11H/班)   ← 注意用11H，非11.5H
                       × 单件台时 ÷ 稼动率

能耗费用（元/件）    ≈ 设备功率（KW）× 能耗利用率 × 单件台时 × 电价（元/KWh）
                     （实测值比理论高~7.7%，差值原因见 open_questions.md Q3）
```

### 3.5 固定参数

| 参数 | 值 | 备注 |
|------|-----|------|
| 工时单价 | 24 元/H | 固定不变 |
| 电价 | 1 元/KWh | 可调 |
| 年生产天数 | 300 天 | |
| 折旧方式 | 10年 2班 折90% | 固定公式 |
| 班产量用班时长 | 11.5 H/班 | 仅用于班产量计算 |
| 折旧用班时长 | 11 H/班 | 仅用于折旧计算 |
