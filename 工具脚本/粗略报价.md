# 注塑粗略报价规则（Decision Tree）

> 用途：快速估算注塑零件的综合单价，驱动自动报价 agent。
> 结构：三大成本模块独立计算，最终汇总加利润得出报价。

---

## 总公式

```
单件报价 = (材料费 + 模具刀具摊销 + 加工费) × (1 + 利润率)
所有的文件都必须要读一遍
```

---

## 一、材料价格

> 目标：算出每个零件的材料成本（元/件）

### 1.1 输入条件
- 材料牌号 / 类型
- 零件克重（g）
- 水口料比例（%）

### 1.2 材料识别规则

**从图纸读取材料信息，按以下优先级处理：**

```
情况A：图纸注明材料名称 + 牌号（常见）
  → 例：PC / MAC5210 / 金发
  → 直接 match 材料价格表，取对应单价

情况B：图纸只注明材料名称，无牌号（少数）
  → 客户需要我们推荐材料
  → 客户会提供物性表（材料物性要求）
  → 根据物性表向厂家询价，不用价格表
  → 报价备注：材料价格待厂家确认

情况C：复合材料（如 部分PC，部分ABS）
  → 将两个材料都写上去
  → 逐一 match 价格表中对应牌号
  → 取实际使用牌号的单价

```

**材料价格表匹配逻辑：**
1. 先按完整牌号精确匹配
2. 匹配不到 → 按材料大类模糊匹配，标注"价格待确认"
3. 均无法匹配 → 触发情况B流程

### 1.3 材料单价来源
→ 引用外部表格 `材料价格表.xlsx`（用户定期更新，以最新版为准）

### 1.4 计算公式
```
材料费（元/件）= 净用量（kg）× 材料单价（元/kg）× (1 + 废品率)
净用量 = 单件实际耗料（含水口，从图纸/称重获取）
```

### 1.5 净用量估算

**Step 1：从STP文件算成品重量（必须做，误差<1%）**
```python
# 工具：cascadio + trimesh
cascadio.step_to_obj(stp_path, obj_path)
mesh = trimesh.load(obj_path)
vol_cc = abs(mesh.volume) / 1000        # mm³ → cc
product_weight_kg = vol_cc * density / 1000
```
密度从材料规格书取（如 PA66+CF10 = 1.16 g/cc）

**Step 2：加水口料 → 净用量**
```
净用量 = 成品重量 × (1 + 水口料比例)
```

**水口料比例经验值（持续积累）：**
| 条件 | 水口比例 | 案例 |
|------|---------|------|
| 热流道 | 0~2% | — |
| 冷流道，大件厚壁 | 15~25% | 金榕CHASSIS 286×253×82mm → 20% |
| 冷流道，扁平薄件 | 30~45% | 金榕AP COVER 287×253×17mm → 40% |
| 冷流道，一般件 | 10~20% | 待积累 |

⚠️ 薄件水口料比例极高（流道料可能比产品本身还重），不可低估。

### 1.6 水口料比例

由模具设计（流道布局、浇口数量）和产品特性（外形尺寸、壁厚、进浇位置）共同决定，纯经验判断。

**影响因素：**
- 热流道 → 水口料极少（≈0~2%）
- 冷流道 + 产品小 → 水口料占比高（可达 20~30%）
- 冷流道 + 产品大 → 水口料占比低（5~15%）
- 多穴模 → 流道更长，水口料占比相对更高

## 一·五、外观工艺分析（出价前必做）

> 目标：判断外观等级，决定模具钢材、晒纹成本、浇口位置、废品率。每个零件都要做。

**① 外观件 vs 内饰件**
- 判据：技术协议外观要求强度（有无「分型面不可见」「色泽/纹理符合样板」「无熔接痕/缩痕」）+ 安装位置（外露 or 隐藏）。
- 外观件 → 型腔高抛/晒纹、优质模具钢（718H/S136）、浇口必须避开可见面（侧浇口/潜浇口/热流道）、外观检验严。
- 内饰件/隐藏件 → 要求宽松，普通钢，浇口位置自由。

**② 是否有皮纹（晒纹）**
- 判据：模具计划有无「上皮纹/晒纹」工序、BOM 的 Texture Type、技术协议纹理要求。
- 有皮纹 → 模具增加晒纹（蚀纹）费用（数千~数万/副）；皮纹面脱模斜度需加大（一般 ≥3°）；影响顶出设计。

**对报价的影响（必须落到数字）：**
| 外观等级 | 模具钢/工艺 | 废品率 | 浇口 |
|---------|-----------|--------|------|
| 外观件 + 皮纹 | 优质钢 + 晒纹 | 1.5~2% | 避开可见面 |
| 外观件 无皮纹 | 优质钢 + 高抛 | 1~1.5% | 避开可见面 |
| 内饰/隐藏件 | 普通钢 | 0.5~1% | 自由 |

⚠️ 不得对外观件用裸模价 + 默认 1% 废品率蒙混。外观+皮纹的模具费和废品率都要上调，并在估值 call-out 中标注依据。

## 二、模具刀具费用

> 目标：算出每个零件分摊的模具成本（元/件）

### 2.1 输入条件
- 产品复杂度
- 模具穴数
- 预计总产量（寿命内）
- 是否含热流道

### 2.2 模具总价
→ **外部询价**，拿到供应商报价后填入，不在本系统估算。

### 2.3 摊销公式
```
模具摊销（元/件）= 模具总价（元）÷ 分摊总件数

分摊总件数 = 客户总采购量（件）
           （若采购量 > 模具寿命内最大产量，则取模具寿命内最大产量）
```

**示例（金榕框架前）：**
- 模具总价：21.98 万元，穴数 2，寿命 30 万模
- 客户采购：23 万件/件号 × 2 = 46 万件 < 60 万件（寿命上限）
- 单件摊销：219,766 元 ÷ 460,000 件 = 0.478 元/件

### 3.1.1 穴数优化 Simulation（必须跑）

**规则：任何产品报价前，必须跑穴数优化，不得直接拍定穴数。**

**⚠️ 例外：穴数已被客户/技术协议指定时，穴数锁定，不做优化。**
- 若询价资料（技术协议、模具要求、BOM 等）已明确写出模具配置（如「1+1，一副模具」「2+2」等），则穴数由甲方设计钉死，**没有优化自由度**，直接采用指定穴数，不跑 simulation。
  - 例：GEA轮罩技术协议 2.4「气坝前左+前右 1+1，一副模具」→ 锁定 2 穴（LH+RH 各 1），单模件数 G30=2，不优化。
- 只有在**穴数由我方自行决定**（客户未指定）时，才必须跑 optimizer 找最优穴数。
- 不论哪种情况，报价前都要先确认「穴数是谁定的」：先在所有文件里找有没有指定，找到→锁定，没找到→跑优化。

**逻辑：**
- 穴数↑ → 模具成本↑、机台吨位↑（能耗/折旧↑）
- 穴数↑ → 每件人工↓、每件折旧↓、每件能耗↓（摊薄）
- 两者之间存在最优点，靠simulation找出

**Simulation步骤：**
```
for N in [1, 2, 4, 6, 8, 12, ...]:
    1. 所需锁模力 = N × 单穴投影面积 × 型腔压力 / 1000 × 安全系数
    2. 选机台 = 向上取最近可用吨位（从设备表）
    3. 若机台不存在（超出最大1300T）→ 跳过
    4. 模具总价 = f(N)（见下方估算规则）
    5. 计算每件总成本
    6. 记录结果
取总成本最低的N为推荐方案
```

**模具总价与穴数关系（经验估算，持续积累）：**
| 穴数 | 相对单穴模具成本倍数 |
|------|-----------------|
| 1 | 1.0× |
| 2 | 1.4× |
| 4 | 1.8× |
| 8 | 2.5× |

**案例验证（Lucid地板舱盖，总量875,000件）：**
- 1穴/60T vs 4穴/250T → 4穴更优（人工摊薄效果远大于模具和机台成本上升）
- 正确答案采用4穴/250T，模具7万 ✓

### 3.1.2 家族模 vs 分机排产

同一批询价中，若有多个零件，需判断是否合并到同一台机器（家族模）。

**优先考虑家族模的条件（满足越多越倾向合并）：**
- 零件尺寸接近，投影面积相当
- 壁厚接近（周期时间差异小）
- 材料相同（同一料筒，无需换料）
- 批量小（分摊机台成本的压力大）

**倾向分机排产的条件：**
- 壁厚差异大（厚件主导周期，薄件被迫等待，浪费产能）
- 材料不同（必须分机）
- 合并后吨位跳档到更大机台，机台费反而更高

**吨位计算规则：**
```
家族模：锁模力 = Σ 所有腔投影面积 × 型腔压力 ÷ 1000 × 安全系数
分机：  每台独立计算
```

**报价时的处理：**
- 默认先算家族模方案
- 若合并后吨位跳档或壁厚差>1mm，再算分机方案做对比
- 取总成本更低的方案，并在报价备注中说明

### 3.2 成型周期（加工时间）

**影响因素：** 壁厚、材料、模具冷却设计
**经验公式（参考）：**
```
冷却时间（s）≈ 壁厚²（mm）× 2.5（PP基准，其他材料系数不同）
成型周期 ≈ 冷却时间 × 2（注射+保压+开模约占一半）
```
→ **Open Question**：见 `open_questions.md` Q2

### 3.3 班产量
```
班产量（件/班）= 11.5 × 3600 ÷ 成型周期（s）× 穴数
```

### 3.4 各项计算公式

```
单件台时（H）        = 成型周期（s）÷ 3600

加工工时（H）        = 单件台时 × 操作人数 ÷ 稼动率

人力费用（元/件）    = 工时单价 × 加工工时
                     = 24 × 单件台时 × 操作人数 ÷ 稼动率

折旧费用（元/件）    = 设备价格（元）× 0.9
                       ÷ (10年 × 300天 × 2班 × 11H/班)   ← 注意用11H，非11.5H
                       × 单件台时 ÷ 稼动率

能耗费用（元/件）    ≈ 设备功率（KW）× 能耗利用率 × 单件台时 × 电价（元/KWh）
                     （实测值比理论高~7.7%，差值原因见 open_questions.md Q3）
```

| 电价 | 1 元/KWh | 可调 |

## 四、汇总与利润

```
单件成本 = 材料费 + 模具摊销 + 加工费
单件报价 = 单件成本 × (1 + 利润率) × (1 + 17.65%)
```

**运输费说明：**
运费目标 = 最终总价的15%，倒推得出在成本基础上加 17.65%（= 15% ÷ 85%）。
- 验证：base × 1.1765 = final，运费 = final × 15% ✓
