# 注塑粗略报价规则（Decision Tree）

> 用途：快速估算注塑零件的综合单价，驱动自动报价 agent。
> 结构：三大成本模块独立计算，最终汇总加利润得出报价。

---

## 总公式

```
单件报价 = (材料费 + 模具刀具摊销 + 加工费) × (1 + 利润率)
```

---

## 一、材料价格

> 目标：算出每个零件的材料成本（元/件）

### 1.1 输入条件
- 材料牌号 / 类型
- 零件克重（g）
- 水口料比例（%）

出材率 = 1 - 水口料比例

## 三、加工工序费用

> 目标：算出每个零件的生产加工成本（元/件）

### 3.1 构成
- 机台费（注塑机折旧 + 能耗）
- 人工费
- 辅助设备费

### 3.2 输入条件
- 机台吨位
- 成型周期（秒）
- 穴数
- 班次/自动化程度

### 3.3 计算公式
```
加工费（元/件）= 机台小时费用（元/h）× 成型周期（s）÷ 3600 ÷ 穴数
```
