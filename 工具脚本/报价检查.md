# 报价检查清单

踩过的坑，每次报价前对照检查。

---

- [ ] **1. 所有文件必须全部读完**
  询价文件夹里每一个文件都要读，不得跳过。
  如果有文件暂时读不了（格式不支持、加密等），必须明确说明哪个文件没读、原因是什么。

- [ ] **2. 读完所有文件后，必须确认每个产品的以下三项**
  - **体积**：从 STP 文件计算（cascadio + trimesh），不得估算
  - **密度**：从材料物性表提取，不得用理论计算或假设值代替。
    - 图纸/BOM有牌号 → 物性表由用户保证提供，直接取密度
    - 无牌号（情况B）→ 走客户推荐流程，物性表待定，报价暂停材料费计算
  - **材料种类 + 牌号**：BOM 与规格书双重核对，两者不一致时必须说明

- [ ] **3. 多零件询价时，判断家族模 vs 分机排产**
  材料相同、尺寸/壁厚接近 → 优先考虑家族模；材料不同或壁厚差>1mm → 分机。不得默认每个零件各用一台机器。

- [ ] **5. 外观工艺分析（每个零件必做）**
  - **① 外观件 vs 内饰件**：依据技术协议外观要求 + 安装位置（外露/隐藏）+ 有无「分型面不可见 / 符合样板」语句判断。
  - **② 是否有皮纹**：依据模具计划「上皮纹/晒纹」工序 + BOM Texture Type + 技术协议纹理要求判断。
  - 外观件 → 模具用优质钢、浇口避开可见面、废品率上调（≥1.5~2%）；有皮纹 → 模具加晒纹费、加大脱模斜度。必须计入模具费和废品率，不得用裸模价/默认1%蒙混。

- [ ] **4. 年产量 / 生命周期量 / 分摊量，必须取邮件或用户口述，不得用图纸/BOM 的量**
  **优先级：用户当面口述 > 询价邮件 > 图纸/BOM。** 图纸或 BOM 里的年产量优先级最低，不能拿来算产值或分摊。
  - 邮件的询价表通常逐零件给：单车用量、**年需求**、生命周期量、**3年分摊数量**——这几个是核算总价和产值的唯一依据。
  - 模具费分摊量按邮件标注的「3年分摊数量」，不是图纸量、不是生命周期量。
  - 算年产值 = Σ（每个零件号的年需求 × 该零件含税单价）；零件号要逐个数，不要把多个零件号合并成「套」再除以年限。
  - 踩过的坑（GEA轮罩）：误用模板示例量 22.96万、又把 6 个零件号当「62万套÷8年」推算，年产量算成 31万件；邮件实际是每个零件号 76,522/年 × 6 = 45.9万件/年，产值差了近一倍。
